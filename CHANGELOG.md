## [1.0.0-beta] - 2026-02-03

### Added
- Initial release
- Manager dashboard with real-time analytics
- Driver management system
- AI-powered order import (Google Gemini)
- Route optimization algorithm
- Live fleet tracking map
- Mobile apps (iOS/Android via Capacitor)
- Supabase authentication and database
- Row-Level Security (RLS) policies

### Features
- **Authentication:** Email/password with role-based access
- **Orders:** CRUD operations, AI import, geocoding, bulk actions
- **Drivers:** Profile management, online/offline status, activity logs
- **Maps:** Real-time tracking, route visualization, GPS integration
- **Optimization:** Nearest-neighbor + 2-Opt algorithms
- **Dashboard:** Live metrics, driver performance, activity feed
